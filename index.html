How to Play

1. Set your bet amount using the + and - buttons or quick bet options
2. Click the SPIN button to start the game
3. Match 3 symbols to win! The more valuable the symbol, the bigger the win
4. Match 2 symbols for smaller wins
5. Hit the 🍑 jackpot for the MEGA PRIZE!

Payout Structure

• 3 Matching Symbols: 3x symbol value
• 2 Matching Symbols: 2x symbol value
• Jackpot Symbol (🍑): Progressive Jackpot + Bonus

Special Features

• Auto Spin: Automatically spin multiple times
• Turbo Mode: Faster gameplay experience
• Max Bet: Instantly set maximum bet amount
• Progressive Jackpot: Grows with every spin!
